خطوات إنشاء المستودع ورفع الموقع على GitHub Pages:

  1. إدخال اسم المستودع:

    • في خانة "Repository name"، اكتب اسمًا مناسبًا مثل:
      quiz-web-app أو educational-quiz.

  2. اختيار نوع المستودع:

    • تأكد من تحديد "Public" حتى يكون الموقع متاحًا للجميع.

  3. إنشاء المستودع:

    • اضغط على زر "Create repository" في أسفل الصفحة.

رفع ملفات الموقع إلى المستودع:

  1. بعد إنشاء المستودع، ستظهر لك صفحة تحتوي على أوامر Git، لكن يمكنك رفع الملفات يدويًا.
  2. اضغط على "Upload an existing file".
  3. اسحب ملف HTML الذي قمت بإنشائه (الذي يحتوي على الاختبار) وقم برفعه.
  4. اضغط "Commit changes" لحفظ الملف في المستودع.

تفعيل GitHub Pages للحصول على رابط الموقع:

  1. انتقل إلى إعدادات (Settings) المستودع.
  2. في القائمة الجانبية، اضغط على "Pages".
  3. تحت قسم "Branch"، اختر الفرع الرئيسي (main) واضغط "Save".
  4. ستظهر رسالة تحتوي على رابط الموقع بعد لحظات، ويكون بالشكل التالي:
    arduino
    https://yourusername.github.io/quiz-web-app/
    (استبدل yourusername باسم مستخدم GitHub الخاص بك).

<script>
const questions = [
{ question: "ما هو هدف التعلم التعاوني؟", options: ["تحقيق الفردية", "تنمية المهارات الاجتماعية", "زيادة الضغط على الطلاب", "إهمال التعاون"], correct: 1 },
{ question: "ما هي استراتيجية KWLH؟", options: ["أسلوب للقراءة الفعالة", "طريقة لحل المشكلات", "تقنية حديثة للتدريس", "منهج رياضي جديد"], correct: 0 },
{ question: "ما هو أحد عناصر استراتيجية التعلم النشط؟", options: ["الاعتماد على الحفظ فقط", "مشاركة المتعلمين بفعالية", "عدم التفاعل مع المحتوى", "إلغاء استخدام الأنشطة"], correct: 1 }
];

function loadQuestions() {
const container = document.getElementById("questionsContainer");
questions.forEach((q, index) => {
let div = document.createElement("div");
div.classList.add("question");
div.innerHTML = `<p>${index + 1}. ${q.question}</p>`;
q.options.forEach((option, i) => {
div.innerHTML += `<label><input type='radio' name='q${index}' value='${i}'> ${option}</label><br>`;
});
container.appendChild(div);
});
}

function submitQuiz() {
let score = 0;
questions.forEach((q, index) => {
let selected = document.querySelector(`input[name='q${index}']:checked`);
if (selected && parseInt(selected.value) === q.correct) {
score++;
}
});
document.getElementById("result").innerText = `نتيجتك: ${score} من ${questions.length}`;
document.getElementById("result").style.display = "block";
}

window.onload = loadQuestions;
</script>
